# 实现两栏布局方案

## 目标
将 collect.html 文件中第 920-945 行的两个 `custom-section` 区域转换为电脑端并列的两栏布局。

## 方案

### 1. 结构修改
- 在两个 `custom-section` 区域外添加一个容器 div，使用类名 `custom-columns-container`
- 保持两个 `custom-section` 区域的内部结构不变

### 2. CSS 样式修改
- 为 `.custom-columns-container` 添加 flexbox 布局
- 设置 `display: flex` 和 `gap: 20px` 以创建两栏布局
- 为每个 `.custom-section` 设置 `flex: 1` 以平均分配宽度
- 在移动设备上（768px 以下）保持单列布局

### 3. 具体修改步骤
1. 在第 920 行前添加容器 div
2. 在第 945 行后闭合容器 div
3. 在 CSS 部分添加 `.custom-columns-container` 的样式定义
4. 在响应式媒体查询中添加相应的样式调整

## 预期效果
- 电脑端：两个区域并排显示，各占 50% 左右宽度
- 移动端：保持原有单列布局
- 保持所有现有样式和功能不变
