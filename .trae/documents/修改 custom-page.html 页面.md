# 修改 custom-page.html 页面计划

## 1. 主色调调整为黄色
- 将页面中的蓝色主色调 (`#1C64F2`) 替换为黄色系颜色
- 修改相关元素：主标题颜色、步骤标题左侧装饰条颜色、链接颜色等

## 2. 第一步内容分两栏展示
- 为第一步内容卡片添加两栏布局
- 使用 Flexbox 或 Grid 实现响应式两栏布局
- 确保在移动设备上自动切换为单栏

## 3. 视频播放区全宽度
- 修改视频容器样式，使其宽度为 100%
- 调整视频播放器尺寸，确保在全宽度下正常显示
- 优化响应式设计，确保在不同设备上都能良好显示

## 4. 重新设计播放器以支持对象存储链接
- 分析现有的对象存储链接格式
- 确保播放器能够正确加载和播放对象存储中的视频
- 可能需要调整 iframe 配置或使用其他播放器方案

## 5. 其他优化
- 确保所有修改后的样式保持一致性
- 测试页面在不同设备上的显示效果
- 验证视频播放器能否正常工作
