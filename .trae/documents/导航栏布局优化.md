# 导航栏布局优化方案

## 电脑网页端修改
1. 修改 `.navbar-container` 样式，将 `justify-content: space-between` 改为 `justify-content: center`
2. 添加 `gap` 属性，为 logo 和菜单项之间设置适当的间距
3. 调整 `.navbar-menu` 的 `gap` 值，确保菜单项之间有合理间距

## 移动端修改
1. 修改 `.navbar-mobile-menu` 样式，设置 `width: 50%`
2. 添加 `margin-left: auto` 使其靠右侧展示
3. 调整 `padding` 和 `border-radius` 等属性，确保视觉效果良好

## 实现步骤
1. 定位导航栏相关的 CSS 样式代码
2. 对电脑端导航栏进行居中布局修改
3. 对移动端汉堡菜单进行宽度和位置调整
4. 测试修改后的效果，确保在不同设备上都能正常显示
