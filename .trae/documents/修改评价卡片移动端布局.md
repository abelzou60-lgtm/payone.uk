## 修改评价卡片移动端布局

### 问题分析
当前代码中，评价卡片在不同屏幕宽度下的布局设置如下：
- 屏幕宽度 ≤ 768px：一行显示2个卡片（flex: 0 0 calc(50% - 15px)）
- 屏幕宽度 ≤ 480px：一行显示1个卡片（flex: 0 0 100%）

### 解决方案
修改CSS代码，移除小屏幕设备（≤480px）的特殊布局设置，使其继承768px以下的布局规则，从而确保所有移动端设备都保持一行两个卡片的展示效果。

### 具体修改
1. 定位到CSS部分的媒体查询：`@media (max-width: 480px)`（879-883行）
2. 修改该媒体查询中的`.review-card`样式，将`flex: 0 0 100%`改为与768px媒体查询中相同的`flex: 0 0 calc(50% - 15px)`
3. 这样修改后，无论屏幕宽度是多少，评价卡片都会保持一行两个的布局

### 预期效果
- 移动端设备（包括小屏幕手机）将始终显示一行两个评价卡片
- 卡片间距和其他样式保持不变
- 桌面端布局不受影响，仍然显示一行四个卡片
