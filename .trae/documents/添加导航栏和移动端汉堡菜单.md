# 添加导航栏和移动端汉堡菜单

## 实现步骤

1. **在 `<body>` 标签后插入导航栏结构**
   - 包含 Logo（使用提供的 URL）
   - 桌面端水平菜单：首页、全球收款、跨境支付、评价
   - 移动端汉堡菜单按钮

2. **添加导航栏 CSS 样式**
   - 固定在页面顶部
   - 桌面端：菜单项作为不同颜色的按钮
   - 移动端：汉堡菜单样式和动画效果
   - 响应式设计（使用媒体查询）

3. **添加 JavaScript 功能**
   - 汉堡菜单的展开/收起逻辑

4. **调整现有样式**
   - 确保导航栏不与现有内容重叠
   - 保持整体设计一致性

## 设计细节

- **Logo**：使用提供的 `https://cf.aibomart.com/payone.uk/payone-logo.png`
- **菜单项链接**：
  - 首页：当前页面
  - 全球收款：`https://payone.uk/colloct`
  - 跨境支付：`https://blog.aibochinese.com/payment/apply%20V-cards`
  - 评价：`https://payone.uk/reviews`
- **按钮颜色**：采用与风车叶片对应的颜色方案
  - 首页：红色
  - 全球收款：蓝色
  - 跨境支付：绿色
  - 评价：橙色
- **移动端**：点击汉堡图标显示/隐藏菜单，菜单项垂直排列
